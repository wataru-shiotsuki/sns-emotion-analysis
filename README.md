# SNS感情遷移分析システム

テレビ局との企業連携PBLとして取り組んだ卒業論文研究です。災害時のSNS投稿から、報道判断につながる情報を抽出するシステムを構築しました。

## 課題
災害報道では、事実確認から実際の報道までの間に、人々の関心が時間とともに変化します。SNS投稿を感情分類するだけでは、この変化や背景要因を報道判断に活かすことができませんでした。

## アプローチ
当初は被害地域をSNSの投稿頻度から可視化する案を検討しましたが、既存サービスと重なり、技術起点の発想自体が課題だと気づきました。そこで報道現場の業務フローを見直し、人々の関心が被害情報から復旧状況へ時間とともに移るという仮説のもと、災害特有の感情を時系列で分類する独自の体系を定義しました。

## 主な機能
- 災害特有の感情分類体系の定義
- SNS投稿の感情分類・時系列可視化
- 8つの災害事例に共通する感情推移パターンの抽出
- 放送倫理に配慮した表現への言い換え機能

## 技術スタック
- Python
- Gemini 2.5 Flash API（感情分類・背景要因抽出）
- Chart.js（時系列可視化）

## 成果
約数万件規模のX投稿を解析し、8つの災害事例に共通する感情の時間的推移を基準として捉え、そこから外れる動きを、支援の遅れや情報不足などその災害特有の問題を示すシグナルとして活用できる形に落とし込みました。
